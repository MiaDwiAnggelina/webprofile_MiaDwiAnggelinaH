About Me

Halo, Saya Mia Dwi Anggelina Hinting, seorang mahasiswi Teknik Informatika di Universitas Palangka Raya Kalimantan Tengah. Saya saat ini berusia 19 Tahun. Website ini adalah tempat di mana saya dapat berbagi perjalanan pendidikan dan minat saya di dunia teknologi. Saya ingin menginspirasi dan berkontribusi pada komunitas IT melalui berbagai pengalaman belajar dan proyek. Saya tertarik pada bidang Web Development dan berusaha memahami lebih dalam tentang Web Development. Saat ini, saya terlibat dalam Studi Independent di Infimite Learning dalam bidang Web Development.

Studi Independent ini membantu saya mengembangkan keterampilan dalam bidang Web Development. Selain studi, saya juga aktif dalam organisasi Dewan Perwakilan Mahasiswa Fakultas Teknik Universitas Palangka Raya menjabat sebagai Sekretaris Umum dan organisasi Persekutuan Sivitas Akademika Kristen menjabat sebagai Koordinator Departemen Sekretariat. Terima kasih telah mengunjungi profil saya. Mari bersama-sama menjelajahi dan mengembangkan dunia teknologi.
